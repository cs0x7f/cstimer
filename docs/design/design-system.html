05 Design Principles

Earn Every Pixel

No decorative elements without function. Borders carry status information. Colors encode meaning. Spacing creates hierarchy. If something doesn't help the cuber make better decisions, remove it.

Warm, Not Cold

This is a practice tool, not a surveillance dashboard. Warm neutrals over blue-greys. Saffron over electric cyan. The interface should feel like a well-worn practice notebook, not a command center.

Data at a Glance

Stat cells use large serif numerals for instant readability. Labels are small monospace above or below. Status dots provide quick scan without reading. The hierarchy is: number → status → detail.

Quiet Confidence

Animations are smooth and purposeful. Hover states reveal, not distract. The interface is confident enough to use restraint. Premium is found in the details: the 1.5px corner accents, the tabular numerals, the grain overlay.
